3) Se crea la funcion Rastrigin

se agregan las lineas: 

float rastrigin(float x, float y){
  return 20
       + (x*x - 10*cos(TWO_PI*x))
       + (y*y - 10*cos(TWO_PI*y));
}

esta es la funcion de rastrigin desarrollada ya que se trabaja en dos dimensiones, osea n = 2

La función recorre cada pixel de la ventana y calcula el valor de la función en ese punto.

se agregan las lineas:


void drawHeatMap(){
  
  // se cargan los píxeles de la pantalla
  loadPixels();
  
  // se recorren las coordenadas en horizontal -->> i
  for(int i=0;i<width;i++){
    
    // se recorren las coordenadas en vertical -->> j
    for(int j=0;j<height;j++){
      
       // la funcion esta definida por x_i en [-3 , 7]
      float x = map(i,0,width,xmin,xmax);
      float y = map(j,0,height,ymin,ymax);
      
      // se evalúa la función rastrigin en ese punto
      float v = rastrigin(x,y);
      
      // El valor de la función se convierte en un color
      float col = map(v,0,80,0,255);
      col = constrain(col,0,255);
      
      // Luego se asigna ese color al pixel (width = ancho de la pantalla)
      pixels[j*width+i] = color(col,0,255-col);
      
    }
  }
